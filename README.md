캘린더 기반으로 개발 프로젝트를 기록하고 회고할 수 있는
**개발 일지 관리 웹 사이트**입니다.

## 📌 프로젝트 소개

**Project Diary**는 개발자가 프로젝트별로
- 일일 작업 내용
- 트러블슈팅 과정
- 회고 내용

을 기록할 수 있도록 만든 웹 서비스입니다.

Fullcalendar 기반의 캘린더 UI를 통해
날짜별 개발 기록을 한눈에 확인할 수 있습니다.

---

## ✨ 주요 기능

- **로그인 / 사용자별 데이터 관리**
- **Fullcalendar 기반 개발 일지 캘린더**
- **프로젝트별 일지 관리**
- **일지 작성 / 수정 / 삭제**
- **트러블슈팅 기록 (문제 / 해결 과정)**
- **이번 달 통계 요약**
  - 작성한 일지 수
  - 트러블슈팅 횟수
  - 진행 중 프로젝트 수
- **Lazy Loading & 병렬 데이터 로딩으로 성능 최적화**
